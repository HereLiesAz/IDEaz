# Static Site template

A zero-build static website: plain HTML, CSS, and JavaScript.

- **IDEaz preview:** works as-is — no build step.
- **Deploy:** serve the folder from any static host. For GitHub Pages, enable
  Pages → "Deploy from a branch" (root). No build required.

## Files
- `index.html` — markup and entry point.
- `styles.css` — styles.
- `main.js` — behavior (classic script, no modules/bundler).
